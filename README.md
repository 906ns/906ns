# みあ

長崎の製造業で5年ほど働きながら、独学でPythonをやってます。
2026年中に福岡に移って、バックエンドエンジニアか業務自動化系の仕事に就きたいと思っています。

副業でせどりをやっていて、確定申告が毎年10時間以上かかる作業だったのでPythonで自動化しました。自分の白色申告に実際に使っています。仕事でも同じように、現場の手作業をコードで消していきたいと思っています。

## よく使う技術

Python / FastAPI / Flask / SQLAlchemy / PostgreSQL / SQLite
Claude API / Gemini Vision API / Groq
Playwright / Selenium / Tesseract OCR
Docker / Docker Compose / Raspberry Pi
Git / GitHub
